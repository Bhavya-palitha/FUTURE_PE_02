# AI UGC Ad Content (Task 2)

This project demonstrates persona-based prompt engineering to generate high-converting UGC ad content for different user types.

## Concept
This project uses persona-based segmentation to create targeted ads based on different user psychology.

## Personas Covered
- Insecure Student (Emotional targeting)
- Influencer Type (Confidence & appearance)
- Busy Student (Time-saving solution)

## Features
- Scroll-stopping hooks
- UGC-style ad scripts
- Conversion-focused CTA
- Platform-specific content (Instagram Reels)

## Tools Used
- ChatGPT
- Prompt Engineering Techniques

## Outcome
Created realistic, engaging, and conversion-focused ad content using AI.
